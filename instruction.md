# Run Instructions

## Prerequisites
- .NET SDK 10.x

## Run
1. From the project root, run:
   - `dotnet run`
2. Open the site:
   - https://localhost:44341/
3. Open the Umbraco backoffice:
   - https://localhost:44341/umbraco

## Notes
- The SQLite database is included at `umbraco/Data/Umbraco.sqlite.db`.
- If the app is already running, stop it before rerunning (`Ctrl+C`).
